egeneralov.postgresql-repository
================================

Postgresql repository provision

Requirements
------------

[Debian](https://www.postgresql.org/download/linux/debian/) or [Ubuntu](https://www.postgresql.org/download/linux/ubuntu/)

Example Playbook
----------------

    - hosts: localhost
      remote_user: root
      roles:
        - egeneralov.postgresql-repository
